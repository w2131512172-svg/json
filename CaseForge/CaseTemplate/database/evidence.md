## 证据编写规则

每条证据至少包含：

- id
- 名称
- 类型
- 所在地点/来源
- 获取条件
- 初见信息
- 深入调查信息
- 可证明什么
- 不能证明什么
- 关联 NPC / 时间线 / 地点
- 是否结案必要
