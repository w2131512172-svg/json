# Case002 玩家可见证物卡

本目录用于存放玩家调查后可获得的证物文本。

注意：

- 本目录内容原则上可给玩家查看。
- 但 DM 应按调查进度逐张释放。
- 不应在开局一次性给出全部证物。

---

## 证物卡列表

- `E01_door_lock_log.md`：入户门锁记录。
- `E02_safe.md`：书房保险柜。
- `E03_blood_trace.md`：书房血迹与拖痕。
- `E04_phone.md`：手机取证摘要。
- `E05_parking_cctv.md`：地下车库监控摘要。
- `E06_call_record.md`：通话记录摘要。
- `E07_forensics_preliminary.md`：初步尸检摘要。
- `E08_company_index.md`：电脑文件索引残留。
- `E09_financial_record.md`：异常转账记录。

---

## 使用建议

玩家调查对应地点或提出对应申请后，DM 可将对应证物卡内容作为发现结果发给玩家。
