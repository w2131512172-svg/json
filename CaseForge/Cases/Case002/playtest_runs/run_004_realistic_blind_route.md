# Playtest Run 004：模拟真实玩家盲测路线

案件：Case002《第十九层》
状态：DRAFT TEST
路线类型：模拟真实玩家 / 非标准顺序 / 无 DM 文件读取

---

## 0. 测试边界

本次测试假定玩家只读取玩家可见材料：

- `player.md`
- `player_materials/case_brief.md`
- 按调查进度释放的 `evidence/` 证物卡

玩家不读取：

- `secret.md`
- `database/`
- `investigation/flow.md`
- `playtest_runs/`

---

## 1. 测试目标

验证 Case002 在非标准顺序下是否仍然可玩。

重点检查：

1. 玩家是否会自然先被表面案情吸引。
2. 玩家是否能通过现场证据发现异常。
3. 玩家是否会过早陷入单一嫌疑人。
4. DM 是否有足够材料纠偏。
5. 玩家是否能在不读真相库的前提下形成结案链。

---

## 2. 模拟玩家路线

### Step 1：玩家先查三名死者位置，而不是门锁

玩家行动：

> 我先看三个人分别倒在哪里，现场有没有明显搏斗痕迹。

DM 可释放：

- 书房血迹与拖痕摘要。
- 主卧防御性损伤方向。
- 顾念房间手机状态。

测试结果：PASS

说明：即使玩家不先查门锁，也能从三处现场差异中发现“这不是简单同一时间突发杀人”。

---

### Step 2：玩家怀疑家庭矛盾

玩家行动：

> 有没有可能是家庭内部冲突？先查夫妻关系和女儿社交关系。

DM 可释放：

- 家庭关系存在压力，但不足以解释门锁、保险柜和外部路径。
- 校园矛盾线可提供，但需要配套不在场证明。

测试结果：SOFT PASS

说明：家庭/校园线可以作为合理误导，但目前玩家材料中对“家庭关系”描述较少，可能略显空。

建议：后续补一张 `player_materials/social_background.md`，提供不剧透的人际关系背景。

---

### Step 3：玩家转查门锁和监控

玩家行动：

> 如果是外人进来，门锁和监控应该有东西。查门锁、查电梯、查车库。

DM 可释放：

- 门锁记录卡。
- 车库监控卡。
- 保安登记异常。

测试结果：PASS

说明：门锁与监控可以稳定把玩家从“普通盗窃/家庭矛盾”拉回“有权限进入者”。

---

### Step 4：玩家过早锁定现场执行者

玩家行动：

> 这个人机会太明显了，我想直接指控他。

DM 反馈应按 `ending.md` 与 Run 003 处理：

- 允许该方向成立为“现场执行嫌疑”。
- 但提示证据链尚不能解释上游动机、失踪物品、案前诱导和案后掩护。

测试结果：PASS

说明：过早锁定不会直接毁掉案件，因为结案系统能拦截“只抓执行者”的半成品答案。

---

### Step 5：玩家查失踪物品

玩家行动：

> 失踪的不只是钱吧？查保险柜里到底少了什么。

DM 可释放：

- 保险柜证物卡。
- 电脑文件索引残留卡。

测试结果：PASS

说明：玩家可以自然从财物失踪转向“特定物品失踪”。该转向不突兀。

---

### Step 6：玩家查公司线

玩家行动：

> 死者工作单位和近期项目有问题吗？查公司、电脑、通话、财务。

DM 可释放：

- 电脑文件索引卡。
- 财务异常卡。
- NPC 口供差异。

测试结果：SOFT PASS

说明：公司线可以展开，但玩家可见材料仍偏摘要。缺少一张更正式的“公司背景/死者近期异常行为”材料。

建议：新增 `player_materials/company_background.md` 或 `evidence/E11_company_context.md`。

---

### Step 7：玩家申请法医复检

玩家行动：

> 三个人是不是同一时间死的？申请更详细法医复检。

DM 可用 `database/forensics.md` 分层释放。

测试结果：PASS

说明：法医库已经足够支撑死亡顺序推理，但玩家可见证物卡目前只有初检，复检卡尚未独立成文。

建议：新增 `evidence/E10_forensics_review.md`。

---

### Step 8：玩家提交结案

玩家行动：

> 玩家提交包含现场执行、上游动机、伪装目的、死亡顺序、关键证据链的结案陈述。

测试结果：SOFT PASS

说明：案件可以被解出，但目前依赖 DM 从数据库中释放部分补充信息。玩家可见文件数量仍不足以完全独立支撑结案。

---

## 3. 总体结论

```text
Run 004：SOFT PASS
```

Case002 已经具备 DM 主持式试玩能力。

但距离 Release Candidate 还差三类文件：

1. 玩家可见背景材料。
2. 更细的复检/公司线证物卡。
3. NPC 分阶段问话卡。
