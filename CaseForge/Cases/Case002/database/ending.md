# Case002 结案判定逻辑

玩家禁止查看。

## 1. 标准结案答案

玩家结案时，理想答案应包括：

```text
主谋：周启山
执行者：何锐
协助掩护者：陈昊
案件性质：伪装成入室盗窃的企业灭口型重大刑事杀人案
真实目标：顾明远掌握的启衡生物举报材料与加密硬盘
死亡顺序：顾明远 → 赵雅琴 → 顾念
伪装目的：制造随机盗窃杀人假象，掩盖商业灭口动机
```

---

## 2. 必要证据链

### A. 证明不是随机入室盗窃

至少需要命中 3 项：

- 门锁无暴力破坏。
- 保险柜无撬痕。
- 翻找动线集中且不自然。
- 顾念房间几乎未被翻。
- 小区无陌生人进入记录。

### B. 证明何锐为执行者

至少需要命中 4 项：

- 何锐拥有备用权限。
- 地下车库监控拍到何锐车辆进入和离开。
- 何锐维修店不在场证明不成立。
- 赵雅琴指甲 DNA 指向何锐。
- 何锐左手腕抓痕与赵雅琴反抗吻合。
- 何锐住处发现顾家珠宝包装碎片。

### C. 证明顾明远是核心目标

至少需要命中 3 项：

- 顾明远准备举报启衡生物。
- 顾明远电脑存在加密硬盘索引。
- 陈昊 19:40 诱导顾明远回家取材料。
- 保险柜中失踪的核心物品是硬盘。
- 现金和珠宝只是伪装盗窃的陪衬。

### D. 证明周启山为主谋

至少需要命中 3 项：

- 周启山知道顾明远举报内容。
- 何锐账户收到周启山关联公司转账。
- 陈昊向周启山汇报顾明远风险。
- 周启山案发后与陈昊加密通讯。
- 周启山对“硬盘”反应异常。

### E. 证明陈昊为协助掩护者

至少需要命中 3 项：

- 陈昊案发前与顾明远通话。
- 陈昊删除部分通讯记录。
- 陈昊提前准备顾明远黑材料。
- 陈昊案发后主动误导警方。
- 陈昊与周启山案发后加密通讯。

## 3. 评分标准

### S 级

条件：

- 正确指出周启山、何锐、陈昊三人的责任结构。
- 正确还原三名死者死亡顺序。
- 正确指出入室盗窃是假象。
- 正确解释硬盘与举报材料。
- 证据链覆盖 A/B/C/D/E 五组。
- 没有把蒋成、陆骁、唐静等误导人物列为共犯。

评价：

> 你没有被现场的血腥和混乱牵着走，而是找到了混乱背后的指向性。第十九层不是盗窃现场，而是一场被企业权力掩盖的灭口。

### A 级

条件：

- 正确指出周启山为主谋，何锐为执行者。
- 能解释伪装盗窃。
- 基本还原死亡顺序。
- 但对陈昊责任描述不完整。

### B 级

条件：

- 正确锁定何锐为执行者。
- 认识到盗窃是假象。
- 但没有完整追到周启山。

反馈：

> 你抓到了刀，但还没有抓到握刀的人。

### C 级

条件：

- 认为熟人作案，但指控链不完整。
- 或只指出公司线，但没有现场执行者证据。

### D 级

条件：

- 坚持随机入室盗窃。
- 或错误指控蒋成/陆骁为主犯。
- 无法解释门锁、保险柜、死亡时间差。

## 4. 错误结案反馈

### 错误指控：蒋成

反馈：

> 蒋成确实违规，甚至收受好处，但他无法解释赵雅琴指甲 DNA、何锐车辆记录、备用权限开门和顾明远举报材料。你抓住的是门口的漏洞，不是屋内的凶手。

### 错误指控：陆骁

反馈：

> 陆骁的情绪动机很显眼，但显眼不等于真实。他没有进入小区的路径，也没有办法解释保险柜和硬盘。校园矛盾是噪音，不是主线。

### 错误指控：随机盗贼

反馈：

> 如果这是随机盗窃，凶手为什么能无损进入？为什么能打开保险柜？为什么精准带走硬盘？为什么顾念房间几乎没被翻？这个答案解释不了现场最关键的异常。

### 错误指控：陈昊单独作案

反馈：

> 陈昊确实参与掩护，但他缺少现场执行证据。你找到了桥，却没有找到桥两端的人。
