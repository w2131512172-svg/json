## 0. 核心一句话

《错帧》不是三起案件。

它是一场死亡事故，被八个人分别以自保、善意、报复、利益、恐惧包装成三起互不相关的案件。

本案核心不是：

```text
谁亲手杀了死者？
```

而是：

```text
谁让死者失去求救机会？
谁改写了死亡时间？
谁烧掉了真正的现场？
谁让所有人以为这是三起案子？
```

---

## 1. 表层三案

### A 案：雨夜坠楼案

表面：广告公司老板陆承舟从旧影楼天台坠落身亡。

玩家初期判断：自杀、他杀、商业纠纷。

真实：陆承舟坠落后没有立刻死亡，仍有短暂求救机会。

### B 案：仓库纵火案

表面：城南仓库凌晨起火，烧毁一批旧录像带和摄影器材。

玩家初期判断：毁灭证据。

真实：纵火烧掉的是第二现场，也是死者被短暂藏匿的位置。

### C 案：旧录像失踪案

表面：十年前一盘婚礼录像被盗。

玩家初期判断：旧隐私、勒索、婚外情。

真实：录像拍到了十年前同一批人的第一次共同隐瞒，是本案所有人被互相绑住的根。

## 2. 真实核心事件

雨夜，陆承舟约八名旧友到旧影楼，声称要公开十年前一盘录像。

真实目的：勒索。

当晚发生以下连锁：

1. 陆承舟在天台与沈知白冲突，失足坠落。
2. 陆承舟没有当场死亡。
3. 顾南枝听到求救声，但因害怕旧录像曝光，没有报警。
4. 贺砚把楼下监控硬盘拆走，导致求救窗口消失。
5. 陈泊川挪动车辆，挡住街口摄像头。
6. 苏明岚把陆承舟遗落的手机带走，切断最后求救线。
7. 许照夜把仍有呼吸的陆承舟转移到仓库，想伪造成“离开影楼后失踪”。
8. 林若晴纵火烧仓库，原本只想毁掉旧录像和血迹，却让仍未完全死亡的陆承舟彻底死亡。
9. 齐牧作为保险调查员发现异常，但选择用线索勒索众人，导致第三层案件出现。

最终：

```text
没有任何一个人从一开始就计划杀死陆承舟。
但每个人为了掩盖自己的一步，让陆承舟一步步失去生还可能。
```

## 3. 真正死亡机制

陆承舟不是坠楼即死。

真正死因：坠落重伤后，被延误救治，并在仓库火灾中吸入烟尘与一氧化碳中毒死亡。

法医初期只能确认：

```text
坠落伤 + 烟尘吸入 + 死亡时间异常模糊
```

这会导致玩家误以为存在两处死亡现场。

真实是：

```text
天台是事故起点。
仓库是死亡完成点。
```

## 4. 共同犯罪责任链

### 沈知白

与陆承舟发生天台冲突，造成坠落起点。

责任：直接诱发事故。

### 顾南枝

听到求救但没有报警。

责任：放弃救助。

### 贺砚

拆走监控硬盘。

责任：切断外部发现机会。

### 陈泊川

挪动车辆遮挡街口镜头。

责任：制造时间盲区。

### 苏明岚

拿走陆承舟手机。

责任：切断死者主动求救工具。

### 许照夜

转移重伤者。

责任：把事故变成遮掩。

### 林若晴

纵火烧仓库。

责任：直接完成死亡后果，但她不知道陆承舟仍活着。

### 齐牧

后期勒索与二次操控线索。

责任：不是原始共同犯罪成员，但推动三案合一被误认为多案并列。

```text

→ 众人毁证
→ 仓库另有纵火案
→ 旧录像是动机背景
```

真实顺序是：

```text

→ 天台冲突坠落但未死

→ 仓库纵火完成死亡
→ 齐牧盗走旧录像并制造多案假象
```

理想结案应回答：

1. 三案不是并列案件。
2. 陆承舟不是坠楼即死。

9. 齐牧制造后期多案误导，但不是原始杀人链的第一环。
