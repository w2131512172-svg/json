# Run 002：尸检突破路线

状态：SIMULATED

---

## 1. 玩家画像

高敏玩家。

特点：

- 优先查看尸检。
- 不相信发现地点等于死亡地点。
- 会把“异常生理证据”放在口供前面。

---

## 2. 调查路线

### Step 1：优先查看尸检报告

获得：

- F01 尸检摘要。

玩家立即发现：

```text
坠落伤 + 烟尘吸入不能被“坠楼即死”同时解释。
```

形成假设：

```text
陆承舟坠楼后仍活着。
```

---

### Step 2：调查后巷

获得：

- F03 后巷血迹。

玩家发现：

```text
拖痕中断，说明他可能被移动，或自己挣扎移动。
```

---

### Step 3：调查仓库

获得：

- F08 仓库拖痕。
- F09 火源检测。

玩家形成强假设：

```text
仓库可能是死亡完成点。
```

---

### Step 4：追踪跨案物

获得：

- F07 蓝色防雨布。

玩家发现：

```text
旧影楼 → 车辆 → 仓库
```

三案出现物理路径。

---

## 3. 玩家形成的正确方向

```text
陆承舟不是坠楼即死。
坠楼后被移动。
仓库火灾可能完成死亡。
三案不是独立案件。
```

这是正确方向。

但玩家暂时仍不能完整回答：

- 谁切断了求救？
- 手机是谁拿走的？
- 监控是谁拆的？
- 车辆遮挡是谁安排的？

---

## 4. 难度测试

高敏玩家能较早发现三案合一。

这不算失败。

因为本案真正难点不是“三案合一”，而是：

```text
从三案合一推进到共同犯罪责任链。
```

---

## 5. 当前风险

尸检太强。

如果玩家一开始拿到 F01，可能过快排除“坠楼即死”。

但这是公平推理必要入口。

---

## 6. 修正建议

不要削弱尸检。

应增强后续责任链难度：

- 每个 NPC 的行为都可被解释成事后包庇。
- 玩家必须意识到：如果陆承舟未死，这些行为就不是事后包庇，而是死亡链的一部分。

---

## 7. 测试结论

```text
Run 002：PASS
```

高敏玩家可以进入正确方向，但仍需要通过 NPC 问话和证据拼合完成责任链。
