# Run 003：蓝色防雨布路线

状态：SIMULATED

---

## 1. 玩家画像

物证追踪型玩家。

特点：

- 不急着问谁撒谎。
- 优先追踪重复物。
- 会问“同一个东西为什么出现在多个现场”。

---

## 2. 调查路线

### Step 1：调查旧影楼杂物间

获得 F07 的第一部分：

```text
杂物间少了一块蓝色防雨布。
```

玩家初始判断：

```text
可能用于遮雨、包裹、遮挡尸体。
```

---

### Step 2：调查街口停车点 / 陈泊川车辆

获得 F07 的第二部分：

```text
车后备箱发现蓝色化纤纤维。
```

同时获得：

- F06 街口摄像头遮挡。

玩家推理：

```text
防雨布可能进过陈泊川的车。
车辆遮挡不是巧合。
```

---

### Step 3：调查仓库火场

获得 F07 的第三部分：

```text
仓库火场发现烧焦蓝色防雨布残片。
```

同时获得：

- F08 仓库拖痕。
- F09 火源检测。

玩家形成强推理：

```text
蓝色防雨布从影楼出发，经车辆，到达仓库。
```

---

## 3. 玩家形成的正确结构

```text
旧影楼 → 车辆 → 仓库
```

三案不再只是动机相关，而是存在物理移动路径。

---

## 4. 仍然不能直接推出的内容

玩家即使追出防雨布，也暂时不能直接知道：

- 防雨布包裹的是尸体还是重伤者。
- 许照夜是否知道陆承舟还活着。
- 林若晴纵火时是否知道仓库有人。
- 齐牧是否参与原始转移。

因此，防雨布路线不会过早剧透共同犯罪全貌。

---

## 5. 假路线风险

玩家可能形成错误模型：

```text
沈知白杀人后，陈泊川开车转移尸体，许照夜仓库藏尸，林若晴纵火毁尸。
```

该模型能解释：

- 天台争执。
- 车辆遮挡。
- 仓库拖痕。
- 火灾。
- 防雨布。

但解释不了：

- 未发出的求救短信。
- 烟尘吸入意味着火灾时仍有呼吸。
- 多人各自行为的独立动机。

---

## 6. DM 纠偏方式

当玩家说“这是转移尸体”时，DM 应提示：

```text
这条路线可以解释防雨布移动，但目前没有证据证明被移动时陆承舟已经死亡。
```

当玩家说“林若晴毁尸”时，DM 应提示：

```text
火源检测能证明她点火，但不能证明她知道仓库里有人。
```

---

## 7. 测试结论

```text
Run 003：PASS
```

蓝色防雨布可以作为三案合一的稳定入口。

它不会直接暴露全部真相，但能打破三案独立假象。
