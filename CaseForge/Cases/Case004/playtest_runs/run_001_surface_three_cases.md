# Run 001：表层三案路线

状态：SIMULATED

---

## 1. 玩家画像

普通玩家。

默认假设：

```text
三起案件有关联，但应分别找凶手。
```

不知道本案结构包含共同犯罪、蒙太奇、多案合一。

---

## 2. 调查路线

### Step 1：查看开局简报

玩家注意到：

- 坠楼案。
- 仓库纵火案。
- 旧录像失踪案。

初始判断：

```text
陆承舟被杀，仓库火灾用于毁证，旧录像是动机。
```

---

### Step 2：调查旧影楼天台

获得：

- F02 天台栏杆。

玩家推理：

```text
天台有冲突，沈知白嫌疑最大。
```

---

### Step 3：盘问沈知白

沈知白阶段 1 回答：

```text
我确实见过陆承舟，但我离开时他还活着。
```

玩家反应：

```text
典型凶手话术。
```

继续锁定沈知白。

---

### Step 4：调查仓库

获得：

- F08 仓库拖痕。
- F09 火源检测。

玩家推理：

```text
仓库可能是抛尸毁证地点。
林若晴和许照夜有问题。
```

---

### Step 5：调查旧录像

获得：

- F10 W-17 旧录像目录。
- F12 八人旧合照。

玩家推理：

```text
十年前录像是动机，齐牧可能盗走录像勒索。
```

---

## 3. 玩家形成的错误模型

```text
沈知白在天台杀死陆承舟。
许照夜或林若晴协助转移/毁证。
齐牧偷走录像勒索众人。
```

该模型能解释：

- 天台争执。
- 坠落伤。
- 仓库火灾。
- 录像失踪。

但解释不了：

- 肺部烟尘吸入。
- 未发出的求救短信。
- 为什么多个“毁证行为”集中发生在坠楼后不久。

---

## 4. DM 纠偏测试

当玩家锁定沈知白时，DM 不直接否定。

应提示：

```text
沈知白单凶路线可以解释天台争执和坠落伤，但解释不了尸检中的烟尘吸入。
```

当玩家锁定林若晴时，DM 提示：

```text
林若晴纵火路线可以解释仓库火灾，但不能解释陆承舟为什么在坠楼案尸检中出现烟尘吸入。
```

---

## 5. 测试结论

```text
Run 001：SOFT PASS
```

表层三案误导成立。

普通玩家会自然进入：

```text
沈知白单凶 + 仓库毁证 + 齐牧勒索
```

但当前纠偏入口依赖尸检报告。

---

## 6. 风险

如果玩家不主动查尸检，可能长时间停留在沈知白单凶路线。

---

## 7. 修正建议

在开局简报和 clue_board 中保持尸检异常可见。

DM 在玩家调查任意现场后，应提醒：

```text
尸检报告是三个案件唯一共同的官方材料。
```
