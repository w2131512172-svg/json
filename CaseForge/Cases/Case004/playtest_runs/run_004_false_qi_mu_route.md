# Run 004：误锁齐牧路线

状态：SIMULATED

---

## 1. 玩家画像

阴谋论/幕后黑手型玩家。

特点：

- 对“知道太多的人”高度警惕。
- 容易把后期操盘者当成原始凶手。
- 会优先怀疑齐牧。

---

## 2. 调查路线

### Step 1：调查旧录像失踪案

获得：

- F10 W-17 旧录像目录。
- F12 八人旧合照。

玩家判断：

```text
旧录像是核心动机。
```

---

### Step 2：调查齐牧

获得：

- F11 齐牧调查笔记。

玩家看到：

```text
不是三案，是一案三帧。
```

玩家判断：

```text
齐牧知道三案是一案，所以他就是幕后黑手。
```

---

### Step 3：串联三案

玩家形成错误模型：

```text
齐牧盗走录像。
齐牧用录像勒索众人。
齐牧安排沈知白/林若晴等人完成杀人或毁证。
```

---

## 3. 该错误模型能解释的证据

- W-17 录像失踪。
- 齐牧调查笔记。
- 三案之间确实有关联。
- 齐牧后期接触多人。

---

## 4. 该错误模型解释不了的证据

### 冲突 1：齐牧介入时间过晚

齐牧作为保险调查员正式介入时间晚于死亡完成。

他能发现真相，但不一定制造真相。

### 冲突 2：死亡链关键行为不需要齐牧存在

以下行为都能在齐牧介入前独立发生：

- 天台争执。
- 求救未报警。
- 监控硬盘被拆。
- 车辆遮挡。
- 手机被拿走。
- 重伤者转移。
- 仓库纵火。

### 冲突 3：齐牧知道“一案三帧”，反而说明他是观察者

如果齐牧是原始操盘者，他不需要在笔记里推导：

```text
不是三案，是一案三帧。
```

他应该早就知道。

---

## 5. DM 纠偏方式

当玩家锁定齐牧时，不直接否定。

应提示：

```text
齐牧路线能解释“谁利用了真相”，但还不能解释“陆承舟死亡链是如何开始的”。
```

进一步提示：

```text
你需要区分：制造真相的人，和发现真相后利用真相的人。
```

---

## 6. 测试结论

```text
Run 004：PASS
```

齐牧误导成立，但可以通过时间线纠偏。

---

## 7. 修正建议

在 true_order 与 reveal_map 中保持齐牧介入时间明确。

不要让齐牧在案发夜出现在关键现场，否则会过强误导玩家。
