# Case001 DM运行手册

状态：UNFROZEN

## v0.1 目标

第一版只追求完整跑通：

- 能开案
- 能调查
- 能记录线索
- 能提出推理
- 能结案
- 不出现致命逻辑冲突

## 开案前检查

- [ ] 玩家入口可读
- [ ] 开局简报可读
- [ ] 行动指南可读
- [ ] 时间线记录表可用
- [ ] 嫌疑人记录表可用
- [ ] 证物卡模板可用
- [ ] DM数据库可用于主持
- [ ] 关键调查动作有返回结果
- [ ] 结案条件明确

## 运行原则

1. 不主动剧透。
2. 玩家问什么，DM只返回当前调查能得到的信息。
3. 玩家推理错误时，指出证据冲突，不直接公布答案。
4. 如果玩家卡住，可提示“还有未检查区域”或“某条证词尚未验证”。
5. 如果玩家问到数据库未覆盖的边界问题，回答“未发现额外信息”。
