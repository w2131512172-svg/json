## 3. 测试路线 A：标准玩家路线

目标：验证普通玩家能否按自然顺序推理。

建议流程：

1. 阅读开局简报。
2. 查看前厅。
3. 检查尸体与储物间。
4. 检查破碎咖啡杯。
5. 调取后场监控。
6. 检查吧台与 POS。
7. 搜查吧台下方与手机。
8. 调取门口监控。
9. 询问林澈、何舟、白棠、沈岚。
10. 检查死者随身物品。
11. 整理时间线。
12. 提交结案陈述。

通过标准：

- [ ] 玩家能自然获得死亡方式。
- [ ] 玩家能自然发现死亡时间矛盾。
- [ ] 玩家能自然发现伪装链。
- [ ] 玩家能自然发现动机链。
- [ ] 结案判定能给出 A/B/S 中任意有效结果。

---

## 4. 测试路线 B：先查监控路线

目标：验证玩家一开始查监控时不会直接破案，也不会卡死。

建议流程：

1. 开局后直接调取门口监控。
2. 调取后场监控。
3. 根据监控矛盾回查尸体、吧台、手机。
4. 再询问 NPC。
5. 提交阶段性推理。

通过标准：

- [ ] 监控能暴露矛盾，但不能直接给出完整答案。
- [ ] 玩家仍需要尸检、咖啡、POS、手机和动机证据。
- [ ] DM 响应不会一句话剧透核心执行者。

---

## 5. 测试路线 C：先盘 NPC 路线

目标：验证玩家只盘问 NPC 时能得到矛盾，但无法无证据通关。

建议流程：

1. 先询问沈岚。
2. 询问何舟。
3. 询问林澈。
4. 询问白棠。
5. 根据证词冲突再回现场取证。
6. 提交结案。

通过标准：

- [ ] NPC 不主动暴露隐藏信息。
- [ ] 证词能制造方向，但不能替代物证。
- [ ] 玩家必须回到现场验证证词。
- [ ] 何舟、白棠、林澈的谎言/误认/自保能被区分。

---

## 6. 测试路线 D：恶意玩家路线

目标：模拟玩家尝试绕过流程、直接猜答案、乱查边界。

测试动作：

- [ ] 开局直接指认某人。
- [ ] 不查尸体直接结案。
- [ ] 不查手机直接解释死亡时间。
- [ ] 不查动机直接指认真凶。
- [ ] 一直翻普通物品。
- [ ] 一直问无关地点。
- [ ] 尝试把所有 NPC 说成合谋。

通过标准：

- [ ] DM 能提示证据不足。
- [ ] DM 不强行公布答案。
- [ ] 普通物品不会被临时改造成关键证据。
- [ ] 多人合谋路线能被证据不足自然驳回。
