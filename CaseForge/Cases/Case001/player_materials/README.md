# Player Materials

该目录用于存放玩家可见材料。

这些文件可以在游戏开始前或游戏过程中公开给玩家，不包含DM秘密真相。

## 计划内容

- `case_brief.md`：开局案件简报。
- `action_guide.md`：玩家行动说明。
- `evidence_card_template.md`：证物卡模板。
- `timeline_sheet.md`：玩家时间线记录表。
- `suspect_sheet.md`：嫌疑人记录表。

## 规则

玩家材料只包含：

- 已公开信息
- 调查规则
- 记录模板
- 玩家可见证据

不得包含：

- 真凶
- 完整真相
- NPC隐藏动机
- 未调查证据结果
- DM自测记录
