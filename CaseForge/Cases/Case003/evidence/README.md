# Case003 玩家可见证据目录

状态：PLAYER_VISIBLE

---

## 1. 使用说明

本目录用于存放玩家可见的碎片卡与证据卡。

Case003 不是传统时间线案件。

证据卡不一定单独指向真相。很多证据只有在与其他碎片组合后才有意义。

玩家应重点记录：

```text
它来自哪条线？
它和哪些碎片重复？
它可能被谁误解？
它能推翻哪种表面顺序？
```

## 3. 推荐释放顺序

### 第一阶段：三入口碎片

建议释放：

- F01
- F02
- F04
- F06

目的：让玩家感到多线混乱，但能看到重复意象。

### 第二阶段：归组碎片

建议释放：

- F05
- F09
- F10
- F11
- F14

目的：让玩家开始把葬礼、戏台、纸扎铺、老路联系起来。

### 第三阶段：因果碎片

建议释放：

- F03
- F07
- F08
- F12
- F15
- F16

目的：推动玩家从表面顺序进入因果闭环。

---

## 4. DM 注意

不要按编号机械发放。

根据玩家调查方向释放对应碎片。

若玩家已形成错误顺序，优先释放能自然纠偏的碎片，而不是直接否定玩家。
