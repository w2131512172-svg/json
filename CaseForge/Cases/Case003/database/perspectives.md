## Line A：葬礼线

### 视角持有者

秦守义、许桂兰、参与葬礼的镇民。

### 玩家初见版本

槐阴镇一个月内连续办了三场葬礼。前两场已经发生，第三场正在筹备。

镇民说葬礼只是为了“让旧人入土为安”。

### 视角盲区

葬礼参与者多数不知道棺木身份问题，只知道流程被老人们控制。

### 视角误读

他们把葬礼理解为结束旧事，但实际上葬礼本身在制造新的身份固定。

### 真实对应

三场葬礼是旧误认被重新加固、暴露、再封闭的三个阶段。

### 关联碎片

F01、F05、F08、F12、F16

### 与其他线的冲突

与 Line C 的“失踪归来”冲突：如果人已入土，为什么还有归来传闻？

与 Line D 的旧款线冲突：为什么葬礼流程会牵扯旧账？

### 修正方式

玩家需要通过棺木记录、无遗像、祠堂看守人的半句供词来发现葬礼核心不是死亡，而是身份。

## Line B：孩子目击线

### 视角持有者

阿禾，以及传播“死人走路”传闻的镇民。

### 玩家初见版本

多年前，阿禾声称自己看见死人从戏台后走了。镇民把这当成鬼故事。

### 视角盲区

阿禾当时年纪小，不能准确判断对方身份，也不理解戏台后发生过什么。

### 视角误读

他把“被认为已死的人”理解成“死人”。

镇民又把儿童误认加工成诈尸传闻。

### 真实对应

这条线记录的是身份错位后，有人离开或被看见离开的痕迹。

### 关联碎片

F06、F09、F13、F14

### 与其他线的冲突

与 Line A 冲突：如果葬礼确认死亡，为什么有人说看见死者离开？

与 Line E 冲突：阿禾记忆中的衣着可能来自戏班道具，而不是真实衣物。

### 修正方式

玩家需要校正红衣、戏台后门、纸扎铺敲门三组碎片。

## Line C：失踪归来线

### 视角持有者

林小满，以及围绕纸扎铺的传闻。

### 玩家初见版本

林小满在给某个无名对象做纸人。她似乎相信一个死去或失踪的人仍与镇上有关。

### 视角盲区

林小满不知道十五年前真正死亡者是谁。

### 视角误读

她怀疑亲人参与旧案，却误判了亲人的角色。

### 真实对应

她的行动让十五年前身份错位重新浮出水面。

### 关联碎片

F04、F09、F13、F16

### 与其他线的冲突

与 Line A 冲突：葬礼试图确认死亡，林小满却不愿确认。

与 Line D 冲突：她以私人亲情解释旧案，但旧案背后还有公共款项和集体隐瞒。

### 修正方式

玩家需要让林小满的纸人、旧外套、敲门事件与无名尸线相互校正。

---

## Line D：旧款线

### 视角持有者

马成河、秦守义、参与过修桥款事务的老人。

### 玩家初见版本

槐阴镇十五年前修过一座桥，但账目不清，后来没人再提。

### 视角盲区

年轻镇民不知道旧款与两名失踪者有关。

### 视角误读

很多人把十五年前的失踪解释为感情、私仇或外出逃亡，却忽略账本和公共款项。

### 真实对应

旧款线是十五年前冲突发生的真正压力来源。

### 关联碎片

F03、F07、F11、F15

### 与其他线的冲突

与 Line A 冲突：为什么葬礼要牵扯旧账？

与 Line C 冲突：为什么亲属线的私人痛苦会连接到公共款项？

### 修正方式

玩家需要把半截账页、修桥碑缺口、货运司机绕路和旧麻绳归入同一组。

## Line E：旧戏班线

### 视角持有者

杜长根和旧戏班成员。

### 玩家初见版本

十五年前雨夜，戏台散场后镇上发生过异常，但每个人记得的参照物不同。

### 视角盲区

杜长根知道戏台散场流程，却不知道散场后被转移的真正对象是谁。

### 视角误读

镇民把戏服、红绸、锣声等演出元素混进真实目击。

### 真实对应

旧戏班线是校正目击时间和衣着误认的关键。

### 关联碎片

F02、F10、F14、F06

### 与其他线的冲突

与 Line B 冲突：孩子看到的“红衣死人”可能不是死者真实衣物。

与 Line A 冲突：葬礼流程使用的身份叙事与戏台散场后的真实行动不一致。

### 修正方式

玩家需要分清“演出散场”“第二声锣”“戏台后门离开”不是同一个时间点。
