---

## 1. 测试目标

Case003 是碎片环形叙事案件，不是传统时间线案件。

本轮 playtest 重点验证：

1. 玩家能否从多个入口收集碎片。
2. 玩家是否会过早试图整理时间轴。
3. 错误顺序是否自然出现。
4. DM 是否能通过碎片纠偏，而不是直接剧透。
5. 玩家能否从碎片组合进入因果闭环。
6. 结案模板是否足够承载最终答案。

---

## 2. 测试路线

### Run 001：碎片收集路线

### Run 002：错误顺序路线

### Run 003：因果闭环路线

目标：验证玩家是否能从碎片归组进入最终结案。

文件：

- `playtest_runs/run_003_causality_closure.md`

---

## 3. 判定等级

### PASS

路线可运行，玩家有足够材料推进，DM 不需要临场补设定。

### SOFT PASS

路线基本可运行，但有信息密度或节奏问题，需要小补丁。

### FAIL

玩家明显卡死，或必须依赖 DM 临场解释核心设定。
