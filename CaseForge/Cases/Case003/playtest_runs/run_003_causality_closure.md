## 6. 发现问题

### 问题 1：责任层级仍需强化

目前玩家能推到闭环结构，但“谁亲手造成结果 / 谁误以为自己造成结果 / 谁推动闭环”的区分可能仍不够清晰。

### 问题 2：结案前缺少一次“责任压问”环节

Case002 有三方对质感，Case003 当前更偏碎片拼图。后续可加入一场“祠堂对质”或“第三场葬礼前夜对质”作为最终压迫场景。

### 问题 3：S 级结案条件略抽象

需要后续补一份 `player_materials/final_questions.md`，用非剧透问题引导玩家提交完整闭环。
