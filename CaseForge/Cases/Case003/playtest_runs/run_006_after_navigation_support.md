## 1. 测试目标

验证 BUGFIX002 是否解决 Run 005 暴露的导航问题。

重点检查：

1. 玩家是否能更早注意地点关系。
2. 玩家是否能更自然接触旧款线。
3. 新增导航材料是否没有剧透核心答案。
4. 案件高难度是否仍然保留。

## 3. 模拟玩家路线

玩家开局阅读 `case_brief.md` 后，注意到地点组合提示。

玩家仍然先调查传闻和纸扎铺，但比 Run 005 更早转向：

```text
废砖窑 / 老桥 / 镇外老路
```

从而更早获得旧款线相关碎片。

---

## 4. 测试观察

### 观察 1：地点组合提示有效

玩家不会立刻知道答案，但会意识到：

```text
废砖窑、老桥、老路可能是一组。
```

这减少了无意义绕圈。

### 观察 2：旧款线进入更自然

玩家不再完全依赖 DM 临场提醒才进入旧款线。

### 观察 3：未明显降低推理难度

新增材料只告诉玩家“地点可以成组”，没有解释：

- 为什么成组。
- 成组后说明什么。
- 谁承担责任。
- 最终闭环是什么。

因此难度仍来自碎片归组、视角校正和因果重建。

---

## 5. 剩余风险

### 风险 1：地点提示可能让结构型玩家更早进入后期

强玩家可能凭地点组合更快抓到旧款线。

但这属于合理推理奖励，不属于剧透。

### 风险 2：仍缺真实图片

`place_map_notes.md` 只是资源规划，实际地图图像尚未生成。
